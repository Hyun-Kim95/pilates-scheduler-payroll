# Pilates Scheduler & Payroll - 사용 가이드

## 로그인

- 관리자: 초기 계정 `admin@pilates.local` / `admin123` (최초 1회 seed 후 변경 권장)
- 강사: 관리자가 강사 등록 후 해당 강사에 연결된 사용자 계정으로 로그인

## 관리자(센터장) 기능

### 강사 관리
- **등록**: 이름, 캘린더 색상, 수업당 요율(금액 또는 %), 월 기본급, 연락처 입력 후 등록
- **수정/삭제**: 목록에서 수정·삭제 버튼으로 처리

### 회원 관리
- **등록**: 회원명, 연락처, 담당 강사(선택), 메모
- **수정/삭제**: 목록에서 수정·삭제

### 스케줄
- **캘린더**: 주간/월간 뷰로 전환 가능. 강사별 색상으로 슬롯 구분
- **슬롯 등록**: 강사별 수업 가능 시간은 "스케줄" 화면에서 별도 슬롯 등록 API/화면으로 추가 가능 (현재는 API로만 등록)

### 예약
- **예약 등록**: 회원 선택 + 강사·시간(슬롯) 선택. 동일 시간대 최대 6명까지 자동 검증
- **예약 취소**: 예약 목록에서 취소 버튼
- **시간 변경**: API `PATCH /api/reservations/:id/move` 로 슬롯 변경 (인원 제한 재검증)

### 정산
- **정산 계산**: 년월 선택 후 "정산 계산/반영" 클릭 → 해당 월 완료 수업 기준으로 강사별 급여 자동 계산 (수업 횟수×요율 + 기본급), 취소/미완료 제외
- **조회**: 강사별·월별 수업 횟수, 수업료 합계, 기본급, 총 지급액 확인

## 강사 기능

- **스케줄 확인**: 본인 슬롯만 주간/월간 캘린더에 표시
- **본인 스케줄 슬롯 등록**: API로 본인 `instructor_id`로 슬롯 추가 가능
- **예약 확인**: 본인 수업 예약만 목록 조회
- **정산 조회**: 본인 월별 정산만 조회 가능 (관리자가 계산한 후)

## 알림톡 (카카오)

- 예약 확정/취소 시 회원 연락처로 알림 발송 가능 (환경 변수 `KAKAO_ALIMTALK_ENABLED`, `KAKAO_ALIMTALK_API_KEY` 설정 후)
- 리마인더(수업 당일 알림)는 스케줄러/배치에서 `sendReminder` 호출로 구현 가능

## 주의사항

- 비밀번호는 배포 후 반드시 변경
- 정산 계산 후 DB에 저장되므로, 수업 완료 처리(`예약 완료`)를 먼저 반영한 뒤 월별 정산 실행
